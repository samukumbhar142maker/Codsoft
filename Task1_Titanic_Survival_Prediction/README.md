# Surviving the Titanic using Machine Learning Algorithms

This study makes use of machine learning algorithms in order to predict whether or not a passenger survived or perished in the sinking of the Titanic.

In order to do this, I made use of the Titanic Dataset which includes data analysis, data cleaning, and data visualization in order to build the model.

# Tools
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn

# Machine Learning Algorithm
- Logistic Regression

# Results
Accuracy of Model = 79%
