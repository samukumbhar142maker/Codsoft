# Movie Rating Prediction

In this project, the prediction is done to forecast the rating of the movies by considering the features like genre, director, actors, etc.

The data was preprocessed before building the machine learning algorithm.

# Tools & Libraries
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn

# Machine Learning Model
- Linear Regression

# Output
The output is given by predicting the movie ratings based on the movie details.
